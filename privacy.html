Política de Privacidad

En IT Automations, la privacidad de nuestros usuarios es prioritaria. Esta aplicación utiliza la API de TikTok exclusivamente para facilitar la publicación de contenido multimedia creado por el usuario.

  • Datos recolectados: Solo procesamos los tokens de acceso necesarios para la publicación.
  • Uso de datos: No compartimos ni vendemos información a terceros.
  • Retención: Los datos se almacenan de forma segura en una infraestructura privada y se eliminan al revocar el acceso.
